It is a demo web site that demonstrates use of jQuery to create a single-page application (SPA) with the Spring backend. The SPA uses AJAX/JSON requests to query the database.

Used technologies:

  1. HTML/CSS/Bootstrap
  2. jQuery 3
  3. Spring Boot
  4. JSON/REST
  5. MySQL
  6. Spring Data

The source code is available at here. Create the database training-day in MySQL while using a local copy and use the relevant user name and password.
